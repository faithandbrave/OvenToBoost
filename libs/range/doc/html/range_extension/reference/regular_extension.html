The one of the remarkable feature in this library is regular operator designed non-intrusively. In addition to regular operator of the Range Adaptor involved in, this library also provides regular operator extension for Range Adaptors in the Boost Range Library. If this library are integrated into the Boost Range Library, this extension will be as the standard feature.
