none_of

Prototype 

template <class SinglePassRange, class UnaryPredicate>
bool none_of(SinglePassRange& rng, UnaryPredicate pred);

template <class SinglePassRange, class UnaryPredicate>
bool none_of(const SinglePassRange& rng, UnaryPredicate pred);

Namespace

boost or boost::range.

Description

none_of returns false if a predicate is true for a referent in the range. range version of C++11 std::none_of.

Definition

boost/range/algorithm_ext/none_of.hpp

Requirements
  1. SinglePassRange is a model of the Single Pass Range Concept.
Complexity

At most boost::end(rng) - boost::begin(rng) applications of the predicate.
